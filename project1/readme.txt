To compile , on windows use compile.bat in cmd, on linux use sh make.sh on terminal
On bin folder : start rmiregistry (windows)
		rmiregistry(linux)

To run a Peer:
	On windows: 
		1- open cmd and go to folder bin
		2- use runPeer1, or runPeer2, or runPeer3, or runPeer4 to run a peer
	On Linux:
		1- open terminal
		4- use sh runPeer1.sh, or sh runPeer2.sh, or sh runPeer3.sh, or sh runPeer4.sh

To run a App:
	On windows use the different runApp bat files
	On linux use the different runApp sh files
